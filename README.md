# Trabucchi Apartments — GitHub Pages Site (NoLo / Centrale)

A lightweight, luxury‑style website to present your Airbnb listing with an always‑visible **Book now** button.

## Pages included

- `index.html` — home (highlights, gallery, reviews, amenities, location)
- `nolo-neighborhood-guide.html` — complete neighborhood guide (SEO page)
- `getting-around.html` — complete transport guide (SEO page)
- `practical-info.html` — practical info + FAQ (SEO page)
- `milan-family-couples-guide.html` — families & couples guide (SEO page)
- `milan-3-day-itinerary.html` — 3‑day itinerary (families & couples)
- `milan-with-kids-checklist.html` — Milan with kids: 3‑day printable checklist (SEO + print/PDF)
- `404.html` — helpful fallback page

## Quick deploy (GitHub Pages)

1. Create a new repository on GitHub (public is easiest), e.g. `nolo-centrale-apartment`.
2. Upload **all files** from this folder (drag‑and‑drop is fine).
3. Go to **Settings → Pages**:
   - **Source:** Deploy from a branch
   - **Branch:** `main` / root
4. Wait for GitHub to publish. Your site URL will appear in the Pages section.

## Google (free) indexing checklist

- Add the site in **Google Search Console** (URL prefix).
- Verify ownership (HTML file upload or meta tag).
- Submit `sitemap.xml`.

## Customize

- Airbnb link: edit the HTML files and search for `https://www.airbnb.it/rooms/1003542985765959264`
- Photos: replace files in `assets/img/` keeping the same filenames (or update the HTML)
- Style: `assets/css/styles.css`
